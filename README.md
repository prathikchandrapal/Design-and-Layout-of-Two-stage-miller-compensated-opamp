# Design-and-Layout-of-Two-stage-miller-compensated-opamp
A two stage opamp with miller compensation was designed, layout for the same was implemented for Post layout verification with clean DRC/LVS/ERC. Matching, routing symmetry, signal shielding, dummy placement, guard rings, floorplanning and power routing were carefully implemented for optimal layout
